# Foodshare Green Tech 🌿

**Group:** FoodShare Green Tech Team
**Members:** Sameer Chaudhary Tharu, Mahesh Batala, Dhiraj Yadav, Prativa Rai, Rupesh Shah
**Course:** Software Engineering (CMP-N204-0)

A web application to reduce food waste by connecting community members to share surplus food.

## Features (Sprint 1)
- **View Food Items**: Browse available food donations in your area.
- **Post Donations**: Share surplus food by filling out a simple form.
- **Responsive Design**: Mobile-friendly interface with a "Green Teach" theme.

## Quick Start
1.  **Clone the repository**.
2.  **Start the environment**:
    ```bash
    docker-compose up --build
    ```
3.  **Access the App**: Open [http://localhost:3000](http://localhost:3000)
4.  **Access Database**: Open [http://localhost:8081](http://localhost:8081) (PHPMyAdmin)

## Technical Stack
- **Backend**: Node.js, Express
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla), Pug (Templates)

## Project Structure
- `app/`: Backend code (`app.js`, `services/`, `views/`)
- `static/`: Frontend assets (`css/`, `js/`)
- `foodshare-db.sql`: Database schema and initial data
